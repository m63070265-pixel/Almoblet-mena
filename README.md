# المبلط مينا (Almoblet Mena) - خدمات تركيب السيراميك والرخام

موقع إلكتروني تعريفي لخدمات "المبلط مينا"، متخصص في أعمال تركيب السيراميك، الرخام، والجرانيت في مدينة الرياض. تم بناء الصفحة لتكون سريعة، متجاوبة مع كافة الشاشات، ومهيأة بالكامل لمحركات البحث (SEO) لاستهداف العملاء في أحياء الرياض ومناطق تقديم الخدمة مثل حي العارض وحي المِصِيف.

## 🚀 المميزات (Features)
- **تصميم متجاوب (Responsive):** متوافق مع الجوالات، الأجهزة اللوحية، وشاشات الديسكتوب.
- **تحسين محركات البحث (SEO Ready):** مدعوم بأكواد `Schema.org` (نوع HomeAndConstructionBusiness) لتعزيز الظهور في نتائج البحث المحلية (Local SEO).
- **أداء عالي (High Performance):** تصميم يعتمد على مكتبات خفيفة عبر (CDN) لضمان سرعة تحميل الصفحة.
- **تتبع وتحليل (Analytics):** مربوط بأدوات Google Tag Manager و Google Search Console.

## 🛠️ التقنيات المستخدمة (Tech Stack)
- **HTML5** لهيكلة الصفحة.
- **[Tailwind CSS](https://tailwindcss.com/)** (عبر CDN) لتنسيق الواجهات بسرعة ومرونة.
- **[FontAwesome 6.5](https://fontawesome.com/)** للأيقونات.
- **[Animate.css](https://animate.style/)** لإضافة تأثيرات حركية سلسة.
- **Google Fonts** (خط Cairo) لدعم الطباعة العربية بشكل أنيق.

## 📂 هيكل المشروع (Project Structure)
- `index.html`: الصفحة الرئيسية التي تحتوي على تفاصيل الخدمات، أرقام التواصل، وساعات العمل.
- `sitemap.xml`: خريطة الموقع لمساعدة عناكب بحث جوجل في الأرشفة السريعة.

## ⚙️ التشغيل والاستضافة (Deployment)
المشروع عبارة عن صفحات ويب ثابتة (Static Web Pages). لا يتطلب بيئة تشغيل معقدة (Node.js أو غيره). 
- **للمعاينة المحلية:** قم بفتح ملف `index.html` في أي متصفح ويب.
- **للاستضافة:** يمكن رفعه مباشرة وبكل سهولة على خدمات مثل GitHub Pages، Firebase Hosting، أو Cloudflare Pages.
